<h4>Executive Summary</h4>

This memo synthesizes the different approaches to model compression discussed across the corpus. The approaches include the use of Bits Per Character (BPC) as a primary evaluation metric, the application of quantization methods, and the implementation of tokenization efficiency measures. These methods aim to reduce the memory requirements of large language models, making them more efficient and accessible.

<h4>Key Findings</h4>

<h5>Bits Per Character (BPC) as a Compression Metric</h5>

One approach to model compression involves the use of BPC as a primary evaluation metric. BPC measures the cross-entropy between the model’s predicted distribution and the true character distribution. When the predicted distribution aligns perfectly with the true distribution, this cross-entropy formulation becomes mathematically equivalent to lossless compression. This equivalence establishes BPC as a theoretically grounded metric for quantifying how effectively large language models compress given text [Source: Farseer_2025, Chunk: Farseer_2025_chunk_0093].

<h5>Quantization Methods</h5>

Another approach to model compression is the application of quantization methods. GPTQ, an approximate second-order method for quantizing large language models, can accurately compress some of the largest publicly-available models down to 3 bits per value on average with little-to-no accuracy loss [Source: GPTQ_2022, Chunk: GPTQ_2022_chunk_0041]. This method is task-agnostic, as it only uses a tiny amount of randomly-chosen data for calibration [Source: GPTQ_2022, Chunk: GPTQ_2022_chunk_0043].

<h5>Tokenization Efficiency Measures</h5>

Tokenization efficiency measures also play a significant role in model compression. Length-MAX, for instance, reduces Tokens Per Character (TPC) by 14-18% compared to other tokenization methods like BPE, SentencePiece, and WordPiece [Source: LengthMAX_2025, Chunk: LengthMAX_2025_chunk_0023]. This method selects tokens directly based on their value in representing the corpus, avoiding the creation of potentially wasteful intermediate tokens [Source: LengthMAX_2025, Chunk: LengthMAX_2025_chunk_0062].

<h4>Conclusions</h4>

The different approaches to model compression discussed in the corpus aim to make large language models more efficient and accessible. By using BPC as a primary evaluation metric, applying quantization methods, and implementing tokenization efficiency measures, these models can be compressed without significant loss of accuracy. These methods provide promising avenues for the future development and deployment of large language models.
