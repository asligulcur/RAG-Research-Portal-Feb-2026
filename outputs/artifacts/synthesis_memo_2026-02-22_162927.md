<h4>Executive Summary</h4>

This memo provides a synthesis of the various approaches to model compression discussed across the corpus. The approaches include the use of Bits Per Character (BPC) as a primary evaluation metric, the application of quantization methods, and the implementation of tokenization efficiency. These methods aim to reduce the memory requirements of large language models (LLMs) without significantly compromising their performance.

<h4>Key Findings</h4>

<h5>Bits Per Character (BPC) as a Primary Evaluation Metric</h5>

BPC is a quantitative metric used to evaluate the compression efficiency of LLMs. It measures the cross-entropy between the model’s predicted distribution and the true character distribution. When the predicted distribution aligns perfectly with the true distribution, the cross-entropy formulation becomes mathematically equivalent to lossless compression, making BPC a theoretically grounded metric for quantifying how effectively LLMs compress given text [Source: Farseer_2025, Chunk: Farseer_2025_chunk_0093].

<h5>Quantization Methods for Model Compression</h5>

Quantization is another approach to model compression. GPTQ, an approximate second-order method for quantizing large language models, can accurately compress some of the largest publicly-available models down to 3 bits per value on average with little-to-no accuracy loss [Source: GPTQ_2022, Chunk: GPTQ_2022_chunk_0041]. This method is task-agnostic and uses a tiny amount of randomly-chosen data for calibration [Source: GPTQ_2022, Chunk: GPTQ_2022_chunk_0043].

<h5>Tokenization Efficiency for Model Compression</h5>

Tokenization efficiency is another approach to model compression. Length-MAX, a tokenization method, reduces Tokens Per Character (TPC) by 14-18% compared to other tokenization methods such as BPE, SentencePiece, and WordPiece across a range of vocabulary sizes [Source: LengthMAX_2025, Chunk: LengthMAX_2025_chunk_0023]. Length-MAX selects tokens directly based on their value in representing the corpus, avoiding the creation of potentially wasteful intermediate tokens [Source: LengthMAX_2025, Chunk: LengthMAX_2025_chunk_0062].

<h4>Conclusions</h4>

The different approaches to model compression, including the use of BPC as a primary evaluation metric, the application of quantization methods, and the implementation of tokenization efficiency, provide effective ways to reduce the memory requirements of LLMs. These methods allow for the compression of models without significantly compromising their performance, making them practical for use in resource-constrained environments.
